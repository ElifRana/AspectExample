## Prerequisites

* H2 Database

## Installation

```sh
mvn clean package

java -jar .target\UserOperations-0.0.1-SNAPSHOT.jar
```

## Used Technologies

* Spring Boot 2.3.4
* Spring Boot Validation
* Spring Boot Jpa
* Spring Profiles
* H2 Database
* Lombok
* Swagger
* Aspect
